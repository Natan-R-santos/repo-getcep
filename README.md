# 📦 Buscador de CEP

Projeto simples para buscar informações de um CEP utilizando JavaScript puro e consumo de API.

## 🚀 Funcionalidades

* Buscar CEP automaticamente
* Consumir API externa
* Exibir:

  * Rua
  * Cidade
  * Estado
* Tratamento básico de erro
* Interface simples com HTML + CSS

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript (Vanilla JS)
* API: BrasilAPI / ViaCEP

## 📚 Aprendizados

Este projeto foi desenvolvido com foco em:

* Manipulação de DOM
* Uso de `fetch` para requisições HTTP
* Trabalhar com `async/await`
* Tratamento de erros com `try/catch`
* Lógica de programação aplicada

## ▶️ Como usar

1. Clone o repositório:

```bash
git clone <url-do-repo>
```

2. Abra o arquivo `index.html` no navegador

3. Digite um CEP válido (8 números)

## ⚠️ Observações

* O CEP deve conter exatamente 8 dígitos
* Caracteres não numéricos são ignorados automaticamente
